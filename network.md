#	网络知识梳理

##	1.TCP/IP认知

	下一层对上一层而言是透明的，传输在每一层是对等的。

###	链路层
	
	以太网帧格式 
	MTU
	ARP			缓存
	RARP

###	网络层

	IP		首部	分片	
	路由表

	ICMP	报文格式	查询、查错

###	传输层

	UDP		无连接、不可靠。 ip首部字段

	TCP		特点、	
			首部、	
			连接控制	三次握手、四次挥手、同时打开、同时关闭、半关闭
			流量控制	滑动窗口、慢启动、拥塞避免、快速重传和快速恢复
			超时重传（四个计时器）
	
###	应用层
	
	DNS		名字空间、指针查询、缓存

	FTP		了解。控制连接+数据连接

	HTTP	报文	请求响应	状态码

	HTTPS	详细握手过程
